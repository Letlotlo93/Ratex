What is RateX?

RateX is a desktop application that compares Rand Exchange rates to other currencies from different banks. It helps you save time and data by displaying currency rate information from different banking websites on one application. It will notify you about the changes in the market, and show you the lowest rates from major banks. Install, get rates.


What are the main features of RateX?

  • Colour coded indicator indicates an increase or decrease in the exchange rate.
  • Pop-up notifier gives hourly pop up desktop notification
  • Exchange rate calculator converts the amount entered in Rands to selcted currency value
  • History tab shows a summary of previous rates

What platforms does RateX run on?

RateX is platform independent, it runs on Windows and Mac
